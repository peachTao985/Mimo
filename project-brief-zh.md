# 项目简介

SmartSpace Agent 是一个面向手机、米家 IoT、穿戴设备、车家互联与家庭服务场景的全屋智能自治中枢。项目目标是让智能家居从“执行指令”升级为“理解场景、主动协同、持续优化”。

它重点解决三个问题：

1. 智能设备和服务分散，用户需要在多个入口之间切换，控制链路长。
2. 传统规则引擎缺乏复杂场景理解，无法处理跨设备、跨时间、跨空间的真实生活需求。
3. 设备执行后的结果和用户反馈没有形成长期记忆，系统无法持续学习。

项目核心能力包括多模态感知、家庭数字孪生建模、长链推理、多 Agent 协作、设备执行闭环和长期偏好记忆。系统可综合家庭设备状态、环境传感器、位置变化、天气、能耗、日程和历史交互，完成复杂场景下的自动决策。

典型场景包括：

- 用户回家前，根据到家时间自动联动空调、灯光、窗帘和安防状态。
- 根据电价、天气、房间占用和设备功率进行节能调度。
- 结合门磁、人体传感器和穿戴设备信号进行老人异常行为识别与分级提醒。
- 在“睡眠”“办公”“观影”“离家”等模式下自动做跨设备协同。

该项目适合展示大模型在智能硬件生态中的平台级价值，尤其适用于需要长上下文、复杂推理和多智能体自治执行的场景。
