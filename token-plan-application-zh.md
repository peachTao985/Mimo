# Token Plan 申请简介

本项目为 SmartSpace Agent，定位为面向小米生态的全屋智能自治中枢，覆盖手机、米家 IoT、穿戴设备、车家互联与家庭服务等场景。项目解决的核心痛点在于，当前智能家居仍以规则触发和单设备控制为主，难以理解用户的复杂意图，也无法在多设备、多场景之间进行连续决策。例如“我快到家了，把家里调舒服一点”“最近电费异常，帮我找原因”“孩子睡了，后续保持安静”这类需求，往往涉及位置、天气、时间、能耗、设备状态、家庭成员习惯和安全策略，传统自动化方案很难完成。

项目核心逻辑为“环境感知-意图理解-状态建模-链式推理-任务规划-多 Agent 执行-结果反馈”。系统先接入设备状态、环境传感器、位置、日程、历史交互与能耗数据；再由意图识别 Agent 判断用户目标，由数字孪生 Agent 建立家庭当前状态，由推理 Agent 进行长链分析和候选决策排序；规划 Agent 将目标拆分为可执行任务，执行 Agent 调用设备控制接口完成联动，安全 Agent 负责权限与异常拦截，总结 Agent 将结果回写为长期偏好记忆。

项目包含长链推理、多 Agent 协作和长上下文处理，对 token 额度有稳定需求。本次申请额度主要用于支撑复杂场景推理、多模态输入理解、家庭状态持续建模、策略生成、效果评测与反馈闭环优化。
